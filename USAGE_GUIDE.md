# 板块查询功能使用指南

## 🚀 快速启动

### 1. 启动Web应用
```bash
# 激活虚拟环境
source ~/shares/bin/activate

# 启动Web应用
streamlit run app.py
```

### 2. 访问应用
- 浏览器会自动打开 `http://localhost:8501`
- 或手动访问该地址

## 📊 功能使用

### 主页导航
1. 打开应用后，在主页选择 **"板块资金查询"**
2. 点击 **"进入板块资金查询"** 按钮

### 板块查询模式
选择 **"单板块详情"** 模式进行详细查询

## 🔍 两种查询方式

### 方式一：当前详情查询

#### 操作步骤：
1. **输入板块名称**：在左侧输入框输入关键词
   - 例如：`医药`、`科技`、`金融`、`消费`、`能源`

2. **选择匹配板块**：系统显示5个最相似的板块
   - 每个选项显示：板块名称 (指数代码) - 相似度
   - 例如：`创医药 (399275) - 110.0%`

3. **选择查询类型**：选择 **"当前详情"**

4. **点击查询**：点击 **"🔍 查询板块详情"** 按钮

#### 查看结果：
- **板块概况**：成分股数量、上涨下跌统计
- **板块强度分析**：强度评级、强势股比例、资金集中度
- **权重股分析图表**：4个子图全面展示
- **成分股排行榜**：资金流入榜、流出榜、涨幅榜
- **详细数据**：所有成分股的详细信息

### 方式二：历史趋势查询

#### 操作步骤：
1. **输入板块名称**：同上，输入关键词并选择板块

2. **选择查询类型**：选择 **"历史趋势"**

3. **选择历史天数**：
   - 30天：近1个月趋势
   - 60天：近2个月趋势（推荐）
   - 90天：近3个月趋势
   - 120天：近4个月趋势

4. **点击查询**：点击 **"📈 查询历史趋势"** 按钮

#### 查看结果：
- **历史表现汇总**：
  - 累计涨跌幅、累计资金净流入
  - 上涨概率、最大回撤
  - 近7日/30日表现、波动率

- **历史趋势图表**：6个子图展示
  - 涨跌幅趋势、资金流向趋势
  - 换手率趋势、成交量趋势
  - 累计收益曲线、上涨比例趋势

- **详细统计数据**：
  - 收益分析：累计收益、波动率、最大回撤
  - 资金流向分析：净流入概率、累计流向
  - 活跃度分析：换手率、强势/弱势天数

- **历史数据明细**：完整的每日数据表格

## 💡 使用技巧

### 1. 板块搜索技巧
- **精确匹配**：输入完整板块名称，如 `医药生物`
- **模糊搜索**：输入关键词，如 `医药`、`科技`
- **行业分类**：
  - 科技类：`科技`、`软件`、`电子`、`通信`、`互联网`
  - 医药类：`医药`、`生物`、`医疗`、`健康`
  - 金融类：`金融`、`银行`、`保险`、`证券`
  - 消费类：`消费`、`食品`、`饮料`、`零售`
  - 能源类：`能源`、`电力`、`石油`、`煤炭`

### 2. 数据解读技巧

#### 当前详情数据：
- **加权平均涨跌幅**：按权重计算的板块整体涨跌
- **主力资金净流入**：权重股资金流向汇总
- **板块强度**：
  - 极强：强势股比例>50%，涨停股≥3只
  - 强：强势股比例>30%，涨停股≥1只
  - 中等：强势股比例>20%
  - 弱/极弱：强势股比例较低

#### 历史趋势数据：
- **上涨概率**：历史上涨天数占比，>50%为强势
- **最大回撤**：历史最大损失，风险评估指标
- **波动率**：价格波动程度，>2%为高波动
- **累计收益**：整个期间的总收益率

### 3. 投资决策参考

#### 基于当前详情：
- **强势板块**：板块强度"强"或"极强"，资金净流入
- **弱势板块**：板块强度"弱"或"极弱"，资金净流出
- **权重股表现**：关注权重前5股票的表现

#### 基于历史趋势：
- **趋势判断**：
  - 近7日表现 > 近30日表现：短期走强
  - 累计收益为正且波动率适中：稳健上涨
  - 上涨概率>50%：历史表现较好

- **风险评估**：
  - 最大回撤<-10%：高风险
  - 波动率>3%：高波动风险
  - 近期表现与历史表现背离：需谨慎

## ⚠️ 注意事项

### 1. 数据说明
- **实时性**：当前详情为实时数据，历史数据为收盘数据
- **权重融合**：所有数据按指数成分股权重计算
- **数据来源**：主要来自AKShare，部分来自TuShare

### 2. 使用限制
- **网络依赖**：需要稳定的网络连接
- **API限制**：频繁查询可能遇到限制，建议适度使用
- **数据延迟**：历史数据获取需要时间，请耐心等待

### 3. 投资风险提示
- **仅供参考**：所有数据仅供投资参考，不构成投资建议
- **风险自担**：投资有风险，决策需谨慎
- **多维分析**：建议结合多种分析工具做决策

## 🔧 故障排除

### 常见问题：

1. **搜索无结果**
   - 检查关键词拼写
   - 尝试更通用的关键词
   - 确认网络连接正常

2. **数据获取失败**
   - 检查网络连接
   - 稍后重试
   - 尝试其他板块

3. **图表不显示**
   - 刷新页面
   - 检查浏览器兼容性
   - 清除浏览器缓存

4. **历史数据为空**
   - 选择较短的历史天数
   - 尝试其他板块
   - 检查网络连接

### 技术支持：
如遇到其他问题，请检查：
- Python环境是否正确
- 依赖包是否完整安装
- 网络连接是否稳定

## 🎯 最佳实践

1. **查询流程**：
   - 先查看当前详情了解现状
   - 再查看历史趋势了解背景
   - 结合两者做综合判断

2. **数据分析**：
   - 关注权重股表现
   - 对比历史平均水平
   - 注意资金流向变化

3. **风险控制**：
   - 关注最大回撤指标
   - 评估波动率风险
   - 分散投资降低风险
